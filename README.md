# WebSocket Chat Application 💬

This project is a simple real-time chat application built while learning WebSockets and real-time communication between clients and servers.

The goal of this project was to understand how real-time applications work behind the scenes, how clients maintain persistent connections with a server, and how messages can be instantly delivered without refreshing the page.

## Features

* Real-time messaging
* Instant communication between connected users
* Persistent WebSocket connection
* Frontend built with React and Vite
* Backend built with Node.js
* Lightweight and beginner-friendly architecture

## Getting Started

### Clone the Repository

```bash
git clone <repository-url>
cd WEBSOCKET
```

### Install Backend Dependencies

```bash
npm install
```

### Install Frontend Dependencies

```bash
cd client/latest
npm install
```

### Start the Frontend

```bash
npm run dev
```

### Start the Backend

```bash
node app.js
```
